```
// Copyright (c) 2025 Your Name or Organization
//
// @file: example_module.h

// @details:
//   本头文件包含 example 模块的接口声明。
//   提供了对外暴露的类 ExampleClass 以及函数 ProcessData() 等接口。
```

## 函数注释
```
/**
 * @brief 
 *
 * 
 * @param a
 * @param b
 * @return true 
 * @return false 
 *
 * @note
 * @warning 
 */
```

| 标记         | 含义           |
| ---------- | ------------ |
| `@brief`   | 简要说明函数功能     |
| `@param`   | 说明每个参数的含义    |
| `@return`  | 说明返回值的含义     |
| `@note`    | 可选，用于补充说明    |
| `@throw`   | 可选，说明异常情况    |
| `@warning` | 可选，警告使用者注意事项 |
